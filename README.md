This is a recursive repo tracker proyect

## Getting Started

First, install dependencies:

```bash
npm install
```

Second, configure your env:

```bash
user=[user of github account]
repo=[repositorie that you want to track]
```

Third, run the development server:

```bash
npm run dev
```
Enjoy!!
